CoCreate-events

It is used to change values in classes or specific attributes, its name describes its functionality events.

Install

 
npm i @cocreate/events

Or you can use cdn link:

<script>https://cdn.cocreate.app/events/latest/CoCreate-events.min.js</script>

Usage

This is events usage

                                
<article>
<div id="div-01" class="vdom-item">Here is div-01 i should get class hidden
<div>
  <div class="vdom-item" >some content1</div> 
  <span events="test" events-closest=".vdom-item"><i class="fa fa-caret-down"></i></span>
</div>
<div class="vdom-item" >some content2</div>
</div>
<div class="vdom-item" >some content3</div>
<div id="div-03">Here is div-03</div>
</article>
<br>
<hr>        
<div class="vdom-item" events-attribute='class' events="active" events-closest="article > div">some content</div>
<br>
<hr> 
<div class="vdom-item" events-attribute='class' events="active" events-target=".vdom-item">click for active</div>
<br>
<hr>
<div class="vdom-item" id="hey" data-test="" events-attribute='data-test' events="active, other, hi, jean" >some content</div>

This is events usage

Attributes

  • events string required

    It is used to initialize the element, and this attribute is added to the specific attribute if it is defined, by default the class attribute is used

  • events-attribute string optional

    Add or remove values, by default the class attribute is used

  • events-target string optional

    Apply events functionality to a specific element

  • events-closest string optional

    Apply events functionality to a closset element
